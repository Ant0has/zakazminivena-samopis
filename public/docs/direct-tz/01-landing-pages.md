## UTM-метки для Директа

Унифицированный паттерн:
```
?utm_source=yandex&utm_medium=cpc&utm_campaign={CAMPAIGN}&utm_content={KEY}&utm_term={KEYWORD}
```

Где:
- `CAMPAIGN` = `mvn_air_kzn` / `mvn_air_svo` / `mvn_air_ikt` / `mvn_air_mrv` …
- `KEY` = идентификатор группы объявлений (например `kzn_main`, `svo_chr`)
- `KEYWORD` = `{keyword}` шаблон Директа (подставит реальный ключ юзера)

Пример полного URL:
```
https://zakazminivena.ru/airport/kzn?utm_source=yandex&utm_medium=cpc&utm_campaign=mvn_air_kzn&utm_content=kzn_main&utm_term={keyword}
```
