# Яндекс Директ — план запуска

Полный план рекламной кампании для zakazminivena.ru на основе СЯ-исследования
(`seo_research/`), анализа конкурентов (13 доменов) и финального состояния сайта.

**Сайт:** https://zakazminivena.ru
**Регион:** Россия (с фокусом на 9 городов и 14 аэропортов)
**Метрика:** счётчик № <УТОЧНИТЬ — добавить ниже>

## 1. Стратегия и структура

### 1.1 Цели кампании

| Цель | Приоритет | Формула расчёта |
|---|---|---|
| Заявка с формы (`form_submit`) | главная | 1 заявка = 1 макроконверсия |
| Клик в WhatsApp (`whatsapp_click`) | главная | приравнивается к заявке |
| Клик в Telegram (`telegram_click`) | главная | приравнивается к заявке |

| Просмотр карточки маршрута (`view_route_card`) | микро | помогает обучить ML |

### 1.2 Структура кампаний (8 кампаний)

| № | Кампания | Тип | Бюджет/день (старт) | Стратегия |
|---|---|---|---|---|

| 2 | **Аэропорты регионы** (LED/AER/MRV/SIP/KGD/KZN/SVX/OVB/IKT/MMK/AAQ/KRR) | Поиск | 1 000 ₽ | Оплата за конверсии |
| 3 | **Туркластеры** (Карелия/КМВ/Алтай/Байкал/Крым/К.побережье/Калининград/Север) | Поиск | 1 200 ₽ | Оплата за клики (low CTR) |

| 5 | **Услуги/сценарии** (свадьба/корпоратив/делегация/конференция/почасовая) | Поиск | 800 ₽ | Оплата за клики |

**Итого старт:** ~7 500 ₽/день = ~225 000 ₽/мес.

### 1.3 Структура групп объявлений внутри кампании

Каждой посадочной странице соответствует **одна группа объявлений** с уникальными ключами.

Пример для кампании №1 «Аэропорты Москва»:
- Группа SVO — посадка `/airport/svo` + дочерние группы по топ-маршрутам (`/airport/svo/moscow-center` и т.д.)
- Группа VKO, DME, ZIA — аналогично

**Принцип:** одна группа = одна точная посадка. Это даёт алгоритму максимально однородный сигнал и улучшает Quality Score.

## 5. UTM-разметка

**Стандартный шаблон для всех ссылок:**

```
?utm_source=yandex
&utm_medium=cpc
&utm_campaign={camp}  ← airport_moscow / airport_regions / destinations / cities / services / b2b / brand / rsya
&utm_content={ad_group}  ← svo / vko / dme / spb_pulkovo / karelia / kmv ...
&utm_term={keyword}  ← динамический параметр Директа
&yclid={yclid}  ← автоматически проставляется Метрикой
```

**Пример:**
```
https://zakazminivena.ru/airport/svo/moscow-center?utm_source=yandex&utm_medium=cpc&utm_campaign=airport_moscow&utm_content=svo&utm_term={keyword}&yclid={yclid}
```

UTM пробрасывается в форму через скрытые поля и попадает в CRM/email — для атрибуции лидов.

## 7. Дополнительные настройки

### 7.1 Расписание показов

- **Будни:** 7:00–24:00 (показы в пиковые часы прилётов)
- **Выходные:** 8:00–22:00
- **Ночь:** 00:00–06:00 — отключить, кроме «Аэропорты Москва» (ночные рейсы)

### 7.2 Регионы показа

- **Аэропорты Москва:** Москва + МО + регионы прибытия (Россия)
- **Аэропорты регионы:** Россия + страны СНГ (для иностранных гостей)
- **Туркластеры:** Россия + основные города-доноры (Москва/СПб → Карелия и т.д.)
- **Города:** только конкретный город + соседние (для маршрутов «межгород»)
- **B2B:** Россия

### 7.3 Корректировки ставок

| Тип | Корректировка |
|---|---|
| Мобильные | +10% (88% трафика — мобайл) |
| Аудитория «Достигли scroll_to_price без формы» | +50% |
| Аудитория «Достигли формы без сабмита» | +100% |
| Аудитория «Покинули корзину/форму» | +30% |
| Пол: женщины 25–55 (семьи) | +10% |
| Праздничные дни / 31 декабря–10 января | +20% |

### 7.4 Турбо-страницы Директа

**НЕ использовать.** У нас собственные быстрые посадки. Турбо-страницы Директа теряют наш контент.

## 11. Дальнейшие шаги (после запуска)

2. **Неделя 2:** включить «Оплату за конверсии» там, где собралось ≥10 конверсий (нужно для алгоритма).
3. **Неделя 3:** запустить РСЯ-кампанию с ретаргетингом по `scroll_to_price` без сабмита.
4. **Месяц 2:** A/B-тест объявлений (2-й заголовок и текст). Минимум 100 кликов на вариант.
5. **Месяц 3:** масштабировать выигрышные группы (+50% бюджет), отрезать убыточные.
